# 活动信息

- **活动名称**：2026年夏季期间限定海域 前段作战【反撃！第三十一戦隊の戦い】（反击！第三十一战队之战）
- **开始时间**：2026-07-08（随维护更新实装，前段作战同日开始）
- **结束时间**：待公布
- **规模**：前段作战 E1 起，后段作战待公布
- **难度**：甲 / 乙 / 丙 / 丁

## 运营公告要点（2026-07-08）
- 前段作战实装，第一作战海域同步开放。
- **E1：九州沖/南西諸島沖【第三十一戦隊駆逐艦の出撃】**
  - 作战内容：以驱逐舰为主力编成精锐战队，先在南九州近海搜索制压敌潜水部队，再进出南西诸岛海域突入并歼灭敌兵站部队。
  - 投入战力（札）：**「第三十一戦隊」** 与 **「増強第三十一戦隊」** 两种。
  - 可使用**基地航空队**。
  - 随作战推进可**进出佐世保**，并可出击**游击部队（7舰编成）**。
- **E2：南沙諸島沖/オルモック沖/サンベルナルジノ海峡沖【サンベルナルジノ海峡 通峡】**
  - 作战内容：再展开至菲律宾方面。①第三十一战队前路扫荡 → ②多号作战部队向前线输送防卫兵站 → ③舰队主力从圣贝纳迪诺海峡向太平洋方面通峡。
  - 投入战力（札）：**「第三十一戦隊」/「増強第三十一戦隊」**（E1 札可继续用）、**「多号作戦部隊」**、**「連合艦隊」**。
  - 随作战推进**基地航空队可增强**。
- E1 可邂逅：丁型（松型）驱逐「杉」「榧」「竹」、驱逐「梅」「桃」、丁型海防舰、秋月型防空驱逐「冬月」「涼月」等。
- E2 可邂逅：丁型驱逐「樫」、伊号潜艇「伊36」「伊41」（官方明示**下一作战海域也能活跃**）、防空巡洋舰「Atlanta」、战舰「Maryland」「Washington」「South Dakota」等。
- **E2 新舰娘**：South Dakota 级战舰二番舰 **USS BB「Indiana」（印第安纳）**——South Dakota 的妹妹。
- **E2 突破奖励（新舰娘）**：Atlanta 级防空巡洋舰 **USS CL「Reno」（里诺）**——Atlanta 的妹妹，舰队防空战力的核心之一。官方点明她与圣贝纳迪诺海峡及我方伊41潜有接点（史实：Reno 在该海峡附近被伊41雷击重创）。通关 E2 即合流。
- **前段作战三个海域连续实装**（E1–E3 同期开放，E3 为前段决战海域）。
- **E3（前段决战海域）：パラオ沖/ウルシー泊地沖/中部太平洋【ウルシー泊地への本格反攻】**
  - 舞台为帕劳近海→乌利西泊地→中部太平洋（史实中伊36/伊41 回天特攻的目标泊地）。
  - 作战内容：①第三十一战队侦察及前路扫荡（含**强行输送**，成功后基地航空队可前进配备帕劳方面）→ ②**第六舰队（潜艇部队）奇袭**（可发动潜艇基轴的**环礁泊地特殊攻击**；高难度以外可用其他方式攻略）→ ③主力对环礁泊地敢行强袭。
  - 投入战力（札）：「第三十一戦隊」/「増強第三十一戦隊」（沿用）、「第六艦隊」、**「ウルシー攻撃部隊」**（注意：E2 的「連合艦隊」不在列）。
- E3 可邂逅（豪华掉落池）：第百一号輸送艦、**大鳳**、**伊400、伊401、伊13、伊14**（潜特/潜水空母组）、水上机母舰「日進」、**USS BB「Iowa」**、美空母群「Wasp」「Intrepid」「Lexington」「Saratoga」「Langley」等。
- **E3 突破奖励（新舰娘）**：Independence 级航空母舰 **USS「Independence」（独立）**——利用轻巡舰体建造的高速轻空母。官方明示：现阶段反复改装即可改成**夜间作战可能轻空母**，且**后续还预定 if 改装**。

## 难度选择限制（实验性新机制，2026-07-08 公告）
高等级提督若想在 E2/E3 选低难度，有最长 24h 的时间锁；E1 不受影响，所有难度随时可选。

| 提督等级 × 难度 | E2 可选时间 | E3 可选时间 |
|------------------|-------------|-------------|
| **Lv≥100 选丙** | **7/9(四) 12:00 起** | **7/9(四) 22:00 起** |
| **Lv≥80 选丁** | **7/9(四) 12:00 起** | **7/9(四) 22:00 起** |
| 其余情况 | 开战即可 | 开战即可 |
- **新舰娘×2**（均可在 E1 邂逅）：
  - 松型驱逐舰 **「桐」**
  - 秋月型防空驱逐舰 **「花月」**（率领最后的第三十一战队、冠以花名的防空驱逐舰）
