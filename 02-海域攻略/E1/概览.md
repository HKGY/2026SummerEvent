# E1 九州沖/南西諸島沖【第三十一戦隊駆逐艦の出撃】

## 基本信息
- **作战名**：第三十一戦隊駆逐艦の出撃
- **札**：「第三十一戦隊」「増強第三十一戦隊」
- **阶段**：共三条血条 —— **P1（攻坚）→ P2（输送 TP）→ P3（攻坚）**
- **解锁条件**：活动开放即入（前段第一图）

## 通关流程
1. **解谜开 P1 boss**（条件见下）
2. **攻坚 P1**（boss：I 点）
3. **解谜开 P2 boss**
4. **输送 P2**（TP 条）
5. **攻坚 P3**

### 解谜：开 P1 boss
| 条件 | 次数 |
|------|------|
| C2 点 S 胜 | ×2 |
| C3 点 S 胜 | ×2 |
| F 点航空优势 | ×2 |
| H 点到达 | ×1 |

### 解谜：开 P2 boss
| 条件 | 次数 |
|------|------|
| L 点到达 | ×2 |
| 基地（防空）优势 | ×2 |

## 路线与机制
- 主力舰种：驱逐为核心，可用**基地航空队**；作战推进后**进出佐世保**、开放**游击部队（7舰编成）**
- 带路条件（阵容 → 路线）：待补
- 第一段以对潜为主（南九州近海敌潜部队）

| 点位 | 舰种加成 | 分组加成 |

> 倍率按惯例可叠乘（舰种 × 分组 × 点位）。P1 阶段收益最高的是 **A组 DD**（如松/竹/杉/桃/梅、凉月、雪风等：全图 1.03 × I点 DD 1.07 × I点 A组 1.12）；解谜阶段 C2/C3 的 S 胜用 **A组 + DE**（择捉/福江/御藏既在 A组又吃 DE 1.12 全图倍率，对潜也强）效率最好。竹、杉双重分组（A+B），大概率是本图最高倍卡，P1 boss 必带。
