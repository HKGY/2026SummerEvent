# E3 パラオ沖/ウルシー泊地沖/中部太平洋【ウルシー泊地への本格反攻】

> **前段作战决战海域**。前段三海域连续实装，E3 与 E1/E2 同期开放；作战详情（札/编成）待后续公告。

## 基本信息
- **作战名**：ウルシー泊地への本格反攻（对乌利西泊地的本格反攻）
- **舞台**：帕劳近海 → 乌利西泊地近海 → 中部太平洋
- **札**：「第三十一戦隊」/「増強第三十一戦隊」（沿用 E1/E2）、「第六艦隊」（潜艇部队）、「ウルシー攻撃部隊」（新札）
  - ⚠️ **E2 的「連合艦隊」札不在 E3 投入战力之列** —— E3 强袭主力上的是新札「ウルシー攻撃部隊」，E2 连合舰队成员大概率不能直接投入 E3
- **阶段**：推测构成 —— ①第三十一战队侦察/前路扫荡 + **强行输送**（成功后基地航空队可前进配备至帕劳方面）→ ②第六舰队（潜艇部队）奇袭 → ③ウルシー攻撃部隊对环礁泊地强袭
- **解锁条件**：待确认（推测需 E2 通关）
- **突破奖励**：**新舰娘 Independence（Independence 级高速轻空母）**——可改装为夜间作战可能轻空母，后续预定 if 改装；其余难度差异奖励待确认
- **难度选择限制（实验性）**：Lv≥100 选丙、Lv≥80 选丁 → 需等 7/9(四) 22:00 后才可选择本海域难度；甲/乙及低等级提督不受限
- 伊36、伊41（E2 可捞）官方明示可在本海域活跃 → **第六舰队奇袭段确认为潜艇编成出击**。史实上两舰均参加过对乌利西泊地的回天特攻（玄作战·菊水队），大概率有高倍特效。潜艇队伍：伊36、伊41 优先，另备伊47、伊58、伊53 等回天搭载舰
- **环礁泊地特殊攻击**：第六舰队奇袭段可发动以潜艇战力为基轴的特殊攻击（回天特攻致敬机制？发动条件待检证）。官方明示**高难度以外（乙丙丁）可用其他方式攻略**，反过来说甲难度大概率半强制要求潜艇特殊攻击
- **基地航空队前进配备**：完成第三十一战队的强行输送作战后，基地航空队可前进配备至帕劳方面（推测＝缩短半径/增加可达点）
