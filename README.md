# 舰队Collection 2026 夏季活动 攻略合集

![期間限定海域【前段作戦】反撃！第三十一戦隊の戦い](assets/banner.png)

> 活动名称：前段作战【反撃！第三十一戦隊の戦い】＋ 后段作战（拡張作戦）【L'Élan de la Flotte Française -フランス艦隊の躍動-】
> 活动时间：2026-07-08 前段作战开始；**2026-07-19 后段作战开放**（结束待公布，后段预定约一个半月）
> 活动规模：前段 E1~E3 ＋ 后段 E4~E5（E5 为后段最终海域）
> 本文件夹用于整理攻略、配置与个人出击记录，全部以 Markdown 书写。
> **攻略内容默认甲难度**，其他难度差异单独注明。

> ⚠️ **警示：不要把 6-3 的舰队开进活动图！**
> 活动图出击会**贴札锁船**（不可逆）。切编成时务必确认当前不是 6-3（或其他通常海域）的队伍，出击前再核对一遍札与编成。
