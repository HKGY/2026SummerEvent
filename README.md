> 活动名称：前段作战【反撃！第三十一戦隊の戦い】（反击！第三十一战队之战）
> 活动时间：2026-07-08 前段作战开始（结束时间待公布）
> 活动规模：前段作战 E1 起（后段作战待公布）
> 本文件夹用于整理攻略、配置与个人出击记录，全部以 Markdown 书写。
> **攻略内容默认甲难度**，其他难度差异单独注明。

> ⚠️ **警示：不要把 6-3 的舰队开进活动图！**
> 活动图出击会**贴札锁船**（不可逆）。切编成时务必确认当前不是 6-3（或其他通常海域）的队伍，出击前再核对一遍札与编成。
